# CodeAnt AI - Frontend UI

## Overview

**CodeAnt AI - Frontend UI** is a user interface for the **CodeAnt AI** platform, a web application aimed at improving code quality using AI-powered suggestions. The project focuses purely on the **UI** design of the platform, providing a seamless and interactive frontend experience using modern web technologies.

This UI serves as the visual part of the platform, where users can interact with the system, view AI-generated suggestions, and visualize the code analysis process. The project doesn't include AI-powered backend functionality yet but lays the groundwork for future integration.

### Features:
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Intuitive UI**: Easy-to-use interface for interacting with the platform.
- **Navigation**: Smooth navigation between different UI components with **React Router**.
- **Code Display**: Displays code snippets and analysis (future functionality will show AI analysis results).
- **Clean Layout**: Designed with **Tailwind CSS** for quick styling and layout management.

## Technologies Used

- **Frontend**:
  - **React.js**: A JavaScript library for building interactive UIs.
  - **React Router**: For smooth navigation between pages.
  - **Tailwind CSS**: A utility-first CSS framework for styling and layout.
  - **React Icons**: A library to use customizable icons in React applications.

- **Development Tools**:
  - **Node.js**: A JavaScript runtime for building and running the app locally.
  - **NPM**: Node package manager to handle dependencies.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Ankit8239jangid/Demo-Project.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CodeAnt
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
